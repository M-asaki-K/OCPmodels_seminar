# OCPmodels_seminar

当リポジトリは、OCP-modelsによる触媒開発セミナー用の教材をまとめたものです。
まずはマニュアルを参考に、計算環境を構築ください。
その後、ocpmodels_test.ipynbを動かしてみて下さい。正常に計算が完了すればOKです。

本格的な教材は現在準備中ですので、少々おまちください。
